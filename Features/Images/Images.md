# Feature: Images

In this feature we provide the ability to select an image and extract text from that image.

## Views
###ImageView
This file is the main view for all other views, here we can see a visual display of the image and extracted text.

### CameraView
This file allows us to use the default UIKit camera functionality

### ARCCameraView
This view allows the user to capture a photo with a visual overlay of the text in the image

### LiveScannerView
This view is V1 for the ARCCameraView...


## Store
### ImageStore
Where all Image handling takes place
